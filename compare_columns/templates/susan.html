django-vertical-multi-columns

This is a django-vertical-multi-columns EvenVMCView. Data can be read top to bottom, left to right, in your sort order. It is distributed so that the length of each column is roughly the same.
There are two other views ...

  • CriteriaVMCView which lets you provided a set of methods to define what data goes in what column.
  • DefinedVMCView lets you pass pre-defined columns for display.
